# Paper Figure Utilities

This folder contains utilities for regenerating the main TeleopWM paper figures.

## Release-Facing Scripts

- `paper/scripts/compose_rollout_figure.py` — qualitative rollout/action figure.
- `paper/scripts/plot_future_action_eval.py` — future-action MAE and correlation plot.
- `paper/scripts/generate_benchmark_figure.py` — runtime benchmark plot.
- `paper/scripts/generate_sampling_figure.py` — dataset/sampling plot.

Generated figures should be written to:

```text
paper/figures/generated/
```

Only main-paper figure utilities are documented here.
